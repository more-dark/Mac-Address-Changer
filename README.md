## 📌 Features

- ✅ **One‑time MAC change** – quick and easy  
- ✅ **Continuous change mode** – change MAC every X seconds (custom interval)  
- ✅ **Random vendor + random prefix** – fully random MAC addresses  
- ✅ **Beautiful terminal UI** – boxed banner with **Team Dark** (red) and **MAC CHANGER** (blue)  
- ✅ **Automatic interface down/up** – no manual intervention  
- ✅ **Error handling** – invalid interface / permission checks  
- ✅ **Lightweight** – pure Bash, no heavy dependencies

## 🚀 Installation & Usage

### 1️⃣ Clone the repository

```bash
git clone https://github.com/more-dark/Mac-Address-Changer.git
cd Mac-Address-Changer
```
```bash
chmod +x mac-changer.sh
```
```bash
sudo ./mac-changer.sh
```

# 1️⃣ Clone the repository By Name

### Debian/Ubuntu
```
sudo apt install macchanger -y   # Debian/Ubuntu
```

### Arch Linux
```
sudo pacman -S macchanger        # Arch Linux
```

### Fedora
```
sudo dnf install macchanger      # Fedora
```
